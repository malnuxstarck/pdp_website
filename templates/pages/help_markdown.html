    Voici un petit aide-mémoire permettant de vous souvenir des différents éléments disponibles à l’aide de la syntaxe Markdown utilisée (massivement) sur ce site. Sachez que si vous voulez rédiger en HTML, un jeu restreint de balises et d’attributs est également mis à votre disposition.

    La documentation complète du Markdown est disponible ici (en).

    Mise en forme de base

    Titres, paragraphes et sauts de ligne

    Les titres sont définis par le caractère #.

    Les paragraphes sont séparés par une (ou plusieurs) lignes vides. S'il n'y a pas de ligne vide (donc un simple retour à la ligne), le texte ne marquera pas ce retour à la ligne. Pour insérer un retour à la ligne sans faire de nouveau paragraphe, il suffit de terminer la ligne qui précède avec deux espaces.

    Description Vous tapez Vous voyez
    Titre 1 ## texte

    texte
    Titre 2 ### texte

    texte
    Deux paragraphes 
    Ligne 1.
Ligne 2.

Ligne 3.

    Ligne 1. Ligne 2.

    Ligne 3.
    Saut à la ligne 
    Ligne 1.␣␣
Ligne 2.

    Ligne 1.
    Ligne 2.

    Tableaux

    Vous tapez : 

    
| Colonne 1 | Colonne 2 |
|-----------|-----------|
| Donnée a  | Donnée b  |
| Donnée c  | Donnée d  |

    Ce qui donne :

    Colonne 1 Colonne 2

    Donnée c Donnée d

    Code

    Ces différents blocs vous permettent de présenter correctement du code, que ce soit de gros morceaux ou juste pour faire référence, par exemple, à des fonctions en utilisant le bloc de code inline.

    Description Vous tapez Vous voyez
    Code inline `texte` texte
    Bloc de code 
    ```python
from os import path
```

    ou

    ␣␣␣␣:::python

    Formules mathématiques (LaTex)

    Vous avez la possibilité d’incorporer des formules mathématiques dans vos messages et vos écrits de différentes manières.

    Description Vous tapez Vous voyez
    Formule incorporée 
    texte
$\int_0^1 f(x) dx$
suite

    texte $\int_0^1 f(x) dx$ suite
    Formule centrée 
    texte
$$\int_0^1 f(x) dx$$
suite

    texte $$\int_0^1 f(x) dx$$ suite

    Utilisation de formules dans un tableau

    Attention cependant, si vous voulez utiliser des formules mathématiques dans un tableau en utilisant la syntaxe Markdown, dû à un bug de Python-Markdown vous allez devoir échapper les accolades dans vos formules, de la manière suivante : 

    
colonne      | colonne
-------------|---------
$a_\{1\}$    | $a_\{2\}

    Voir cette discussion sur Bitbucket pour plus d’informations.
